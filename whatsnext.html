What's Next

Looking Forward to Grade 10

As we move forward, we're excited to explore new programming languages, face more challenging class projects, and continue developing our creativity and skills in web design and beyond.

We believe that we can expect to learn new coding languages in Grade 10, like Python, and it's sub-language, Pycharm. And with these new languages, we expect that we would also have to commit more of our time into these projects eating up some of the hours we use for our sleep, and replacing it with more work. But learning these languages can also give us an opportunity to bolster our creative web designing skills by giving us a chance to use these new languages and languages we've already learned together, creating new possibilities for web design.
